# WYSIWYG Markdown Editor

![Preview GIF](./public/preview.gif)

Supported style:

1. Inline
   - **bold** / *italic* / ==mark==
   - ~~delete~~ / __underline__ / `code`
   - img ![alt][src]
   - link [innerText][href]
2. Block
   - Heading, support inline heading style only `# Heading 1`
   - Paragraph
   - Hr, support `---` only
   - Checkbox ` - [x] Task1`
   - Blockquote `> `
